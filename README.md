Игра волейбол

Логлайн:
В игре участвуют 2 игрока - player1 (класс P1) и player2 (класс P2), 
между ними расположена сетка. Один игрок подает мяч, второй - принимает.
Цель игры - отбить мяч.

Сеттинг:
Игрок получает 1 очко, если он отобъет мяч.
Если мяч касается земли на стороне противника, то игрок также получает 1 очко.
Если мяч попадает в сетку и касается земли на стороне игрока, то 1 очко получает противник.
